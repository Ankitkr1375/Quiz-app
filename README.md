# Quiz-app
A simple gui based quiz app made using Tkinter 
## Features  
- Interactive GUI built with Tkinter.  
- True false questions
- Score tracking.   
- Lightweight and easy to use.  

## Requirements  
- Python 3.7+  
- Tkinter (usually included with Python by default)
